# Vim 安装指南



```cmd
$ brew install vim --with-lua --with-override-system-vi
```

`lua` 是为了 neocomplete


## 操作

切换视窗 <C w> 方向键

NERDTree
B = 叫出bookmark
C = 把目前游標停留的這個目錄設定為根目錄
p = 把游標移動到上一層目錄
P = 把游標移動到根目錄
J = 把游標移往這個結點的第一個
K = 把游標移往這個結點的最後一個
u = 把樹狀結構的根目錄往上移一層
I = 切換是否顯示隱藏檔案
m = 叫出NERDTree的系統選單
